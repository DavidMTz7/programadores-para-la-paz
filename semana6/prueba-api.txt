Respuesta devuelta por el servidor al enviar la petición POST a http://localhost:3000/registro:

El servidor procesa el objeto JSON entrante usando el middleware "express.json()", mapea los campos extraídos mediante "req.body" e inmediatamente responde con un código de estado 200 OK y el siguiente objeto JSON de confirmación:

{
  "estado": "Datos recibidos",
  "nombre": "Maria",
  "mensaje": "Hola comunidad"
}
